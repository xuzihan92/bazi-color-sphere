一、5/31 模型的问题

5/31 那晚我们把金（白）放在北极、水（黑/深蓝）放在南极。这是从「金=白」「水=黑」的文化标签硬套的，不是从日地关系推导的。

1.1 塌在哪里

  • 金生水：从球顶沿经线下滑→纯白→浅蓝→深蓝→黑。数学上能走，命理上牵强——金（秋分·肃杀）怎么一步滑到水（冬至·极寒）？
  • 火生土：红向球心走→降明度增暖度→沉淀为大地色。但为什么火先到的是北极白极而不是球心的黄？
  • 水在球底（南极黑），金在球顶（北极白）——但火（夏至）和水（冬至）是黄道两端，应该是对跖点关系，不是极点关系。

1.2 漏了什么

把「黑」和「白」当成金和水的「本色」，而不是土的「极态」。黑白是中轴线的两个端点，不是黄道带上四行的属性。

5.2 需修改的核心项

# 问题 v1.0 错误 v2.0 正确做法
1 土的定义 _wuxing_base_color("土", i) → H=45° 土不存在于球面。土的"颜色"是明度值 L——0（南极黑）到 1（北极白）
2 算子数量 五个算子（木火土金水） 四个算子（木火金水）。土不是算子——土是四行算子的极值约束
3 火算子方向 "huo": 195.0（青绿色，错的） 火=夏至，在黄道面高点。HSL 中 0°（红）是合理的——夏至正午的天光色温
4 算子复合顺序 木→火→土→金→水（相生顺序串行） 四行算子沿黄道面运行，土算子只作用于极值（当 S→0 或 L→极值时触发轴收敛）
5 颜色空间 HSL 柱坐标 仍然是 HSL，但 H=黄道面方位角（春0°/夏90°/秋180°/冬270°），L=明度轴（土），S=距轴距离
6 五行扇区 五个扇区等分 黄道面四象限 + 中轴线（球心→两极）

5.3 土算子的正确语义

# 旧（错）：土是一个独立的颜色变换
土(color, i) → 降饱和、向灰偏

# 新（对）：土是"向中轴线收敛"的引力
土作用 = 任何颜色在 S→0 时汇聚到明度轴
       = 当四行算子的强度叠加到极端值时，自动向 L=0.5 收束
       = 土不是主动变换——是四行变换的边界条件

一、天文学验证

1.1 地球轨道与黄道面

地球绕日轨道平面 = 黄道面。四季由黄道面与地球自转轴的倾斜（23.5°）决定：

节气 黄道位置 太阳直射 五行 特征
春分 黄经0° 赤道 木 昼夜等长，生发
夏至 黄经90° 北回归线23.5°N 火 北半球昼最长，极阳
秋分 黄经180° 赤道 金 昼夜等长，肃杀
冬至 黄经270° 南回归线23.5°S 水 北半球夜最长，极阴

关键事实: 夏至与冬至在黄道面上是对跖点（相差180°），春分与秋分也是对跖点（相差180°）。

1.2 v1.0 的天文学错误

v1.0 的球体排布:
        北极（金·白）
            │
   火 ─── 球心(土) ─── 木     ← 火和水在黄道面上不是对跖点!
            │
        南极（水·黑）
  • 火（夏至）和水（冬至）在 v1.0 中相距 90°——但天文学上它们是 180° 对跖关系。
  • 火生土的路径是"向球心走"——但夏至（极阳）向球心（中和）收敛，物理上没有驱动力。
  • 金（白）在北极——但北极在地球上对应的是极昼/极夜，不是任何一个季节的元素。

1.3 v2.0 的天文学自洽

v2.0 的球体排布:
        北极（土·白·阳之极）─ 极昼
              │
    夏至（火） ╱ ╲ 春分（木）
       ╱  黄道面  ╲
      │    土轴     │  ← 中轴倾斜23.5°
       ╲          ╱
    冬至（水） ╱ ╲ 秋分（金）
              │
        南极（土·黑·阴之极）─ 极夜
  • 火↔水：黄道面上 180° 对跖 ✅
  • 木↔金：黄道面上 180° 对跖 ✅
  • 中轴倾斜：23.5° 的黄赤交角解释了为什么"极昼对极夜"不镜像对称——轴本身是斜的 ✅
  • 四行顺时针：火→金→水→木→火 = 夏→秋→冬→春→夏 = 一年四季的自然循环 ✅

天文判决: v1.0 把季节对跖点放错了位置。v2.0 的黄道面排布与地球轨道力学完全一致。

3.2 土的哲学地位

在先秦哲学中，土从来不是和其他四行平级的成员：

3.4 双螺旋的数学描述

用球坐标系描述双螺旋：

设中轴为z'轴（倾斜的），球坐标为 (r, θ', φ')，其中： - r = 到球心的距离 - θ' = 与中轴的夹角（极角） - φ' = 绕中轴的方位角（方位角）

阳螺旋的参数方程：

r(φ') = R·(1 - φ'/(2π))        # 从球面(r=R)向球心(r=0)收束
θ'(φ') = θ₀ + k·φ'              # 极角随方位角线性变化（螺旋上升）
φ' ∈ [0, 2π·n]                   # n = 缠绕圈数

阴螺旋的参数方程：

r(φ') = R·(φ'/(2π))             # 从球心(r=0)向球面(r=R)发散
θ'(φ') = θ₀ - k·φ'              # 极角随方位角反向变化（螺旋下降）
φ' ∈ [0, 2π·n]

当阳螺旋和阴螺旋在球心(r=0)交汇时，整体形成∞形状。

关键参数： - 缠绕圈数 n：理论上为1（一年一圈），但可以有谐波（n=2, 3...对应月、日周期） - 螺距 k：决定螺旋的松紧程度，与黄赤交角23.5°相关 - 交汇点：中5（球心），对应阴阳转换的"枢纽"

4.2 此涨彼消的数学表达

设阳螺旋的"长度"为Y(阳)，阴螺旋的"长度"为Y(阴)：

Y(阳) = sin(ωt)                 # 从冬至(t=0)开始增长
Y(阴) = cos(ωt)                 # 从冬至(t=0)开始衰减

其中：ω = 2π/T，T = 一年周期

验证：
- t=0（冬至）：Y(阳)=0（最短），Y(阴)=1（最长）
- t=T/4（春分）：Y(阳)=0.707，Y(阴)=0.707（等长）
- t=T/2（夏至）：Y(阳)=1（最长），Y(阴)=0（最短）
- t=3T/4（秋分）：Y(阳)=0.707，Y(阴)=0.707（等长）

Y(阳)² + Y(阴)² = 1 —— 阴阳总量守恒！

五、螺旋型色彩映射

5.1 从球面映射到螺旋映射

v2.0的色彩映射是球面映射：

color = f(H, S, L)              # H=黄道方位角, S=距轴距离, L=明度

v3.0的色彩映射是螺旋映射（四维）：

color = f(H, S, L, φ)           # φ = 螺旋相位（时间维度）

螺旋相位φ决定当前处于阴阳螺旋的哪个位置： - φ = 0°：冬至，阳螺旋起点（坎1） - φ = 90°：春分，阳螺旋到达震3 - φ = 180°：夏至，阳螺旋终点（离9） - φ = 270°：秋分，阴螺旋到达兑7

5.2 螺旋色彩渐变

每个卦位（1-9）不是固定的颜色点，而是沿着螺旋路径有色彩渐变：

阳螺旋色彩渐变（1→2→3→4→5）：

坎1（水/黑/深蓝）→ 坤2（土/灰黄）→ 震3（木/翠绿）→ 巽4（木风/青绿）→ 中5（土/灰）
     ↓ 渐亮+变暖          ↓ 渐绿+渐活         ↓ 渐青+渐扬         ↓ 渐收+渐稳

阴螺旋色彩渐变（5→6→7→8→9→1）：

中5（土/灰）→ 乾6（金/银白）→ 兑7（金/肃白）→ 艮8（土/黄褐）→ 离9（火/赤红）→ 坎1（水/黑）
    ↓ 渐白+渐冷          ↓ 渐肃+渐敛         ↓ 渐暖+渐沉         ↓ 渐红+渐烈         ↓ 渐暗+渐寒

5.3 动态色彩场

在任意时刻t，球体上任意点P的色彩为：

C(P, t) = C_base(P) + ΔC_spiral(P, t)

其中：
- C_base(P) = 点P的静态基础色（由五行属性决定，同v2.0）
- ΔC_spiral(P, t) = 螺旋相位引起的动态偏移

动态偏移的计算：
ΔC_spiral = α · sin(φ(P) - ωt) · C_wuxing(P)

其中：
- α = 螺旋振幅（决定动态变化的强度）
- φ(P) = 点P的螺旋相位角（由其在双螺旋上的位置决定）
- ωt = 当前时间相位
- C_wuxing(P) = 点P的五行本色

5.4 八字四柱的四维映射

八字有年/月/日/时四柱，对应四个时间尺度：

柱 时间尺度 螺旋周期 螺旋圈数 色彩影响
年柱 年 T_year = 1年 n=1 主色调，决定基础螺旋相位
月柱 月 T_month = 1月 n=12 月相螺旋，叠加在年螺旋上
日柱 日 T_day = 1日 n=365 日周期螺旋，对应地球自转
时柱 时 T_hour = 1日 n=24 时辰螺旋，最精细的时间尺度

多层螺旋嵌套：

C_total(P, t) = C_base(P) 
                + ΔC_year(P, t)      # 年螺旋（主导）
                + ΔC_month(P, t)     # 月螺旋（次级）
                + ΔC_day(P, t)       # 日螺旋（微调）
                + ΔC_hour(P, t)      # 时螺旋（微调）

这与八字排盘的"层层叠加"逻辑完全一致。

            2: {"name": "坤", "wuxing": "土", "pos": (-0.5, 0.5, -0.5)}, # 西南偏下
            3: {"name": "震", "wuxing": "木", "pos": (-1, 0, 0)},        # 赤道东
            4: {"name": "巽", "wuxing": "木", "pos": (-0.7, 0.7, 0.2)},  # 中轴上端
            5: {"name": "中", "wuxing": "土", "pos": (0, 0, 0)},         # 球心
            6: {"name": "乾", "wuxing": "金", "pos": (0.7, -0.7, -0.2)}, # 中轴下端
            7: {"name": "兑", "wuxing": "金", "pos": (1, 0, 0)},         # 赤道西
            8: {"name": "艮", "wuxing": "土", "pos": (0.5, -0.5, 0.5)},  # 东北偏上
            9: {"name": "离", "wuxing": "火", "pos": (0, 0, 1)},         # 北极
        }

        # 螺旋参数
        self.spiral_turns = 1.0      # 缠绕圈数
        self.spiral_amplitude = 0.3  # 螺旋振幅（色彩偏移强度）

    def get_base_color(self, position):
        """获取静态基础色（同v2.0）"""
        H, S, L = position
        # H = 黄道方位角, S = 距轴距离, L = 明度
        return hsl_to_rgb(H, S, L)

    def get_spiral_offset(self, position, t):
        """
        计算螺旋相位偏移

        Args:
            position: (H, S, L) - 球面坐标
            t: 时间（0=冬至, 0.25=春分, 0.5=夏至, 0.75=秋分）

        Returns:
            (dR, dG, dB) - RGB偏移量
        """
        phi = self._get_spiral_angle(position)    # 点的螺旋相位
        omega = 2 * math.pi                       # 年周期角频率

        # 阴阳螺旋此涨彼消
        yang_strength = math.sin(omega * t)       # 阳螺旋强度
        yin_strength = math.cos(omega * t)        # 阴螺旋强度

        # 计算偏移
        offset = self.spiral_amplitude * (
            yang_strength * self._yang_spiral_color(phi) +
            yin_strength * self._yin_spiral_color(phi)
        )

        return offset

    def get_color(self, position, t):
        """获取动态色彩 = 基础色 + 螺旋偏移"""
        base = self.get_base_color(position)
        offset = self.get_spiral_offset(position, t)
        return base + offset

  6. 黑格尔正反合（ thesis-antithesis-synthesis ）→ 阳螺旋（正）、阴螺旋（反）、球心（合）

  7. 马克思主义否定之否定 → 螺旋上升/下降的周期性辩证运动

  8. 该模型可作为阴阳五行所有分支学科的统一参照系——中医、风水、命理、丹道、历法均可在此几何框架中找到精确对应。

一、引言：问题的提出

1.1 五行色彩的"断裂"之谜

在传统五行色彩理论中，存在一个长期困扰研究者的问题：

木（绿）→ 火（红）→ 土（？）→ 金（白）→ 水（黑）

从木到火，色彩在色环上连续过渡（绿→黄→红）。但从火到土，颜色突然"断裂"——按照v1.0和v2.0之前的理解，火是红色，土是黄色或灰色，金是白色，水是黑色。这种从红→黄→白→黑的跳跃，在二维色环上显得不连贯、不协调。

历代研究者对此给出了各种解释： - "土是中性色，所以是黄/灰"——但这没有解释为什么偏偏是黄/灰 - "金是白色，因为金属反光"——但这是文化类比，不是物理推导 - "水是黑色，因为深渊无底"——同样是类比

这些解释都是后天赋义，而非先天推导。

1.2 梓涵的洞察

2026年6月7日，梓涵提出：

"日晷画出来的太极图正是球体的二维化，而且这个二维化概括的不是单独一个半球不是单螺旋，而是同时概括了一个球体，概括了整个双螺旋的变化。其阴阳此涨彼消的变化正好解释了为什么五行色彩到火生土开始就变成黑白，看起来在色环上似乎和另一半不连贯也不协调，实际上是因为分了纵向的两半，从单独一边看的话，你仔细观察随着时间变化以节气为代表去看影子的长短变化，你会发现越接近火/水影子越短/长，也就越往中央走，到了中央也就正是到了极点白/黑，所以才会表现为黑白连在一块。"

这一洞察的核心是：五行色彩的"断裂"是一个维度压缩造成的假象。 在三维球体中，色彩过渡是连续的；只有在被强行压缩到二维色环时，才会出现"断裂"。

二、太极图作为双螺旋球体的二维投影

2.1 日晷原理与太极图生成

日晷的工作机制： - 一根直立的晷针（中轴）垂直于晷面 - 太阳光照射晷针，在晷面上投下影子 - 随着太阳位置变化，影子的长度和方向随之变化 - 一年中，影子端点在晷面上画出的轨迹就是太极图

关键对应： | 日晷元素 | 双螺旋球体对应 | |---------|--------------| | 晷针 | 中轴（巽4→中5→乾6） | | 晷面 | 黄道面（或赤道面） | | 影子 | 太阳在天球上的投影 | | 影子最短（夏至） | 阳之极（离9/北极白） | | 影子最长（冬至） | 阴之极（坎1/南极黑） | | 影子中等（春秋分） | 阴阳平衡（震3/兑7） |

2.2 为什么太极图同时概括了双螺旋

单螺旋视角（错误）： - 如果只观察阳螺旋（坎1→坤2→震3→巽4→中5），在二维投影中只得到太极图的"阴鱼" - 如果只观察阴螺旋（中5→乾6→兑7→艮8→离9→坎1），在二维投影中只得到太极图的"阳鱼" - 这种理解认为太极图是"一半阴一半阳"的拼接

双螺旋视角（正确）： - 太极图的S形分界线不是"阴鱼和阳鱼的边界" - S形分界线是双螺旋在二维投影中的交汇线 - 阴鱼内部包含了阳螺旋的信息（阳在阴中） - 阳鱼内部包含了阴螺旋的信息（阴在阳中） - 双鱼的眼睛（阳鱼中的黑点、阴鱼中的白点）= 双螺旋在对方领域的"种子"

数学表达：

设双螺旋在三维空间中的参数方程为：

阳螺旋: r_y(φ') = R·(1 - φ'/(2πn)), θ_y(φ') = π/2 - kφ'
阴螺旋: r_i(φ') = R·(φ'/(2πn)), θ_i(φ') = π/2 + kφ'

将其投影到二维平面（沿中轴方向正投影）：

P_y(x, y) = (r_y·cos(φ'), r_y·sin(φ'))
P_i(x, y) = (r_i·cos(φ' + π), r_i·sin(φ' + π))

两条投影曲线在二维平面上恰好形成S形边界，整体构成太极图。

2.3 二维投影的信息损失与恢复

从三维球体到二维太极图的投影会损失一个维度的信息：

维度 三维球体 二维太极图 信息状态
黄道方位角 H ∈ [0°, 360°] 角度 ∈ [0°, 360°] ✅ 保留
距轴距离 S ∈ [0, 1] 半径 ∈ [0, R] ✅ 保留
中轴高度 L ∈ [0, 1] 丢失 ❌ 压缩

丢失的"中轴高度"（L = 明度 = 土的本色）正是五行色彩"断裂"的根源：

  • 在三维中：火（红，L=0.5）→ 向北极收敛 → 土（白，L=1.0）是一个连续的明度渐变
  • 在二维中：L维度被压缩，只剩下H（色相），于是火（红，H=0°）和土（无H，因为是黑白灰）在色环上"断开"

结论：五行色彩的"断裂"不是真实的物理断裂，而是维度压缩造成的信息损失。

三、影子长短变化作为双螺旋阴阳消长的二维度量

3.1 日晷影子的数学描述

设晷针高度为h，太阳高度角为α，则影子长度：

L_shadow = h / tan(α)

一年中影子长度的变化：

节气 太阳高度角 影子长度 双螺旋位置 色彩趋向
冬至 最低 最长 坎1（阴之极） 南极黑
春分 中等 中等 震3（阴阳平衡） 赤道绿
夏至 最高 最短 离9（阳之极） 北极白
秋分 中等 中等 兑7（阴阳平衡） 赤道白偏蓝

3.2 影子长度 = 到中轴的距离

在双螺旋模型中：

影子长度 ∝ 1 / 阳螺旋强度 ∝ 阴螺旋强度

即： - 影子最短（夏至）→ 阳螺旋最强 → 颜色最靠近北极白 - 影子最长（冬至）→ 阴螺旋最强 → 颜色最靠近南极黑 - 影子中等（春秋分）→ 阴阳平衡 → 颜色在黄道面上（四行本色）

影子的长短变化，正是双螺旋阴阳消长在二维平面上的直接度量。

3.3 为什么火生土表现为"黑白"

从日晷视角看：

  • 夏至前后：影子最短，接近晷针根部（中心）
  • 这个"中心"在三维球体中对应北极（白）
  • 所以火（红）→ 土（白）的过渡，在日晷上表现为从边缘红向中心白的收敛
  • 由于二维投影丢失了L维度，观察者只看到"红突然变成了白"

但实际上： - 从三维视角：红（H=0°, S=1, L=0.5）→ 橙红（L=0.6）→ 浅橙（L=0.7）→ 白（L=1.0） - 这是一个连续的明度增加过程，不是断裂

"黑白连在一块"的真相：在三维球体中，黑白从来不是"连在一块"——它们是球体中轴的两端。但在二维太极图中，中轴被压缩成一个点（圆心），所以两极在投影中"相遇"。

五、与哲学框架的统一

5.1 重玄理论（成玄英/李荣）

重玄的核心："玄之又玄，众妙之门"（《道德经》第一章）

  • 第一层"玄"：有（阳螺旋）/ 无（阴螺旋）
  • 第二层"玄"（重玄）：超越有/无，回到"道"本身
  • 在模型中：
  • "有" = 阳螺旋（坎1→中5，从阴到阳的存在展开）
  • "无" = 阴螺旋（中5→离9，从阳到阴的存在收敛）
  • "重玄" = 球心（中5），既非有也非无，既是有也是无
  • "众妙之门" = 球心作为阴阳转换的枢纽，一切变化由此出、由此入

对应关系：

道（无极） = 整个球体（包含双螺旋的全部可能性）
  │
  ├── 有（阳） = 阳螺旋（坎1→中5，存在展开）
  ├── 无（阴） = 阴螺旋（中5→离9，存在收敛）
  │
  └── 玄（球心） = 中5（有无交汇）
       └── 重玄（球心之心） = 中5作为"无无"（连"无"也超越）

5.2 黑格尔正反合（辩证法）

黑格尔三段论： - 正（Thesis）：阳螺旋——存在的肯定、展开、外化 - 反（Antithesis）：阴螺旋——存在的否定、收敛、内化 - 合（Synthesis）：球心（中5）——正反的扬弃（Aufhebung），既保留又超越

螺旋上升： - 黑格尔的"历史螺旋上升"在模型中有精确对应： - 每一次"正→反→合"完成一圈双螺旋 - 但下一圈不是简单重复——螺旋半径变化（r从1→0或0→1），相当于"量变" - 当完成一个完整周期（阳+阴），进入下一周期时，整体水平提升——"质变"

对应关系：

正（肯定） = 阳螺旋：坎1(水)→坤2(土)→震3(木)→巽4(木风)→中5(土)
   │                              存在的展开、生长、外化
   ↓
反（否定） = 阴螺旋：中5(土)→乾6(金)→兑7(金)→艮8(土)→离9(火)→坎1(水)
   │                              存在的收敛、成熟、内化
   ↓
合（扬弃） = 球心中5：既非展开也非收敛，而是展开与收敛的同一
   │                              作为下一周期的起点
   ↓
新的正 = 下一圈阳螺旋：在更高水平上重复

5.3 否定之否定（马克思主义辩证法）

否定之否定的核心：事物发展经过两次否定（肯定→否定→否定之否定），呈现螺旋式上升。

在模型中的精确对应：

  • 第一次否定：阳螺旋（坎1→中5）是对"纯阴"（坎1）的否定——从阴走向阳
  • 第二次否定：阴螺旋（中5→坎1）是对"阳"的否定——从阳回到阴
  • 否定之否定：回到坎1时，不是简单的回到原点——而是完成了一个周期的提升

螺旋式上升：

周期1: 坎1(阴) ─→ 中5(阳) ─→ 离9(阳之极) ─→ 坎1'(阴，但水平提升)
          ↑                    ↓
        否定1               否定2
        (阴→阳)            (阳→阴)

周期2: 坎1'(阴) ─→ 中5'(阳) ─→ 离9'(阳之极) ─→ 坎1''(阴，再次提升)

注意：这里的"水平提升"在模型中体现为螺旋相位的推进（φ增加）

5.4 三哲统一的数学表达

三种哲学在双螺旋模型中统一于同一个几何结构：

哲学 "正" "反" "合/玄" 运动方式
重玄 有（阳） 无（阴） 玄（中5） 有无双运
正反合 正（阳螺旋） 反（阴螺旋） 合（中5） 螺旋上升
否定之否定 肯定（坎1） 否定（中5） 否定之否定（坎1'） 周期性提升

统一公式：

Φ(t) = A·sin(ωt) + B·cos(ωt) = C·sin(ωt + φ)

其中：
- Φ(t) = 任意时刻的"存在状态"
- A = 阳螺旋振幅
- B = 阴螺旋振幅
- C = √(A² + B²) = 总存在量（守恒）
- φ = arctan(B/A) = 螺旋相位

这个公式同时满足：
1. 重玄：A和B的相对大小决定"有/无"的比例
2. 正反合：sin和cos的叠加产生新的质（相位φ）
3. 否定之否定：周期性函数自然产生"肯定→否定→否定之否定"的节律
