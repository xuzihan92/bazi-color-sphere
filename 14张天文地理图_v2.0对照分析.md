## 二、14张图共同揭示的核心规律

### 规律A：地轴倾斜是所有季节变化的根本原因
**来源图**: 3, 4, 11, 12, 14
**规律**: 地轴倾斜23.5° → 太阳直射点在南北回归线之间移动 → 四季变化
**与v2.0对应**: 
- 地轴 = v2.0的"土轴"（中轴线）
- 地轴倾斜 = "土不是垂直的，而是有角度的"——这解释了为什么四行不是均匀分布在赤道面上，而是分布在倾斜的黄道面上
- **新洞察**: v2.0目前将土轴视为"垂直贯穿球心"，但实际上地轴是倾斜的。这意味着"土轴"在v2.0中可能需要引入一个"倾角参数"，使得四行的黄道面与土轴有一个固定的夹角

### 规律B：黄道面是四行运动的舞台
**来源图**: 3, 6, 11, 12, 14
**规律**: 地球绕日公转的轨道平面 = 黄道面。春分/夏至/秋分/冬至是黄道面上的四个关键点。
**与v2.0对应**: ✅ **完全验证**
- v2.0将木(春分0°)→火(夏至90°)→金(秋分180°)→水(冬至270°)定位于黄道面——与图6的24节气公转图完全吻合
- 图12明确标注"黄道面"与"赤道面"的夹角23°27′——这正是v2.0中四行分布的"舞台"

### 规律C：夏至↔冬至 = 对跖点（180°）
**来源图**: 3, 6, 11, 12
**规律**: 夏至与冬至在公转轨道上处于直径两端，相差180°
**与v2.0对应**: ✅ **完全验证**
- v2.0的火(夏至)与水(冬至)在黄道面上相差180°——与图3、图6、图11完全吻合
- v1.0中这两个点仅相距90°——这是v1.0的天文学致命错误

### 规律D：春分↔秋分 = 对跖点（180°）
**来源图**: 3, 6, 11, 12, 14
**规律**: 春分与秋分同样在直径两端
**与v2.0对应**: ✅ **完全验证**
- v2.0的木(春分)与金(秋分)相差180°——与所有公转图完全吻合

### 规律E：太阳直射点做正弦式回归运动
**来源图**: 14（左图）
**规律**: 太阳直射点在一年中从赤道→北回归线→赤道→南回归线→赤道，呈正弦曲线
**与v2.0对应**: 💡 **提供新建模工具**
- 四行算子的"强度"可以用正弦/余弦函数精确建模：
  - 木强度 = cos(θ) · [θ≈0°(春分)最大]
  - 火强度 = sin(θ) · [θ≈90°(夏至)最大]
  - 金强度 = -cos(θ) · [θ≈180°(秋分)最大]
  - 水强度 = -sin(θ) · [θ≈270°(冬至)最大]
- 这为执中的代码提供了**数学基础**：四行强度不是独立的输入参数，而是黄道方位角的函数

### 规律F：昼夜交替 = 地球自转 + 晨昏线
**来源图**: 2, 14（右图）
**规律**: 晨昏线将地球分为昼夜两半球，随着地球自转而移动
**与v2.0对应**: 💡 **阴阳理论的物理对应**
- 晨昏线 = 阴阳分界线
- 昼半球 = 阳，夜半球 = 阴
- 晨昏线的移动 = 阴阳消长
- **新洞察**: v2.0目前只建模了"黄道面四行"（年周期），但还没有建模"昼夜交替"（日周期）。八字中的"时柱"对应的就是日周期——这意味着v2.0需要引入第二个时间维度：黄道面方位角（年周期）+ 地球自转角度（日周期）

### 规律G：五带分布 = 太阳辐射量的纬度梯度
**来源图**: 1, 7
**规律**: 热带(23.5°N-23.5°S)获得最多太阳辐射，寒带(66.5°-90°)最少
**与v2.0对应**: 💡 **土引力的物理对应**
- 热带 = 靠近赤道（中轴平面）= "土"的中和区域
- 寒带 = 靠近两极 = 极端（北极白/南极黑）
- 温带 = 过渡区域 = 四行的活跃区域
- **新洞察**: v2.0中"土引力使颜色向中轴收敛"可以用五带分布来理解：越靠近中轴（赤道）越"土"（中和），越远离越"极端"。这给了执中一个明确的物理直觉来设计"土引力"算法

### 规律H：太阳视运动 = 天球上的倾斜大圆
**来源图**: 5, 8, 13
**规律**: 太阳在天球上的视运动轨迹是一个倾斜的大圆，夏至最高、冬至最低
**与v2.0对应**: ✅ **验证v2.0的球体几何**
- 图13展示的"夏至轨迹""二分轨迹""冬至轨迹"三个平行的大圆——这正是v2.0中"四行在黄道面上运动"的天球投影
- 图8展示的汕头市(23.5°N)太阳高度角：夏至90°(头顶)、春秋分66.5°、冬至43°——这三个角度恰好对应v2.0中"火""木/金""水"在球体上的"高度"

### 规律I：月相变化 = 次级黄道面
**来源图**: 10
**规律**: 月球绕地球公转，月相变化取决于月球相对于地球和太阳的位置
**与v2.0对应**: 💡 **扩展提示**
- 月相周期(~29.5天)是地球公转周期(~365天)的次级周期
- 这提示v2.0未来可以扩展：除了"年周期"的黄道面四行，还可以引入"月周期"的次级球体
- 八字中的"大运"和"流年"可能对应不同时间尺度的"球体嵌套"

### 规律J：南北半球季节相反
**来源图**: 5
**规律**: 北半球夏至 = 南半球冬至，太阳视运动轨迹完全相反
**与v2.0对应**: ✅ **验证"不对称对称"**
- v2.0提出的"南北极不是几何镜像，而是相位对称"——图5的两幅天球图正好展示了这一点
- 北回归线以北：夏至最高、冬至最低
- 南回归线以南：冬至最高、夏至最低
- 这是"相位对称"（相反）而非"几何对称"（相同）的精确天文表达

### 规律K：经纬网 = 球面坐标系
**来源图**: 9
**规律**: 纬线平行、经线汇聚于两极，构成完整的球面坐标系
**与v2.0对应**: 💡 **坐标映射澄清**
- v2.0将H(色相)映射为"黄道方位角"——这对应经度（但经度是0°-360°，H也是0°-360°）
- v2.0将L(明度)映射为"中轴高度"——这对应纬度（但纬度是-90°到+90°，L是0到1）
- v2.0将S(饱和度)映射为"距中轴距离"——这在地理上没有直接对应，但可以用"距赤道距离"类比
- **需要澄清**: HSL的H是经度、L是纬度（但线性映射）、S是距轴距离——这个映射在球面坐标系中是自洽的

## 四、对执中代码的具体建议

基于14张图的规律，对`wuxing_sphere_v2.py`提出以下具体建议：

### 1. 四行强度函数化（基于图14）

```python
import math

def wuxing_intensity(angle_deg: float) -> dict:
    """
    根据黄道方位角计算四行强度
    angle_deg: 0°=春分, 90°=夏至, 180°=秋分, 270°=冬至
    基于图14太阳直射点正弦回归运动
    """
    rad = math.radians(angle_deg)
    return {
        "mu": max(0, math.cos(rad)),      # 春分最大，秋分最小
        "huo": max(0, math.sin(rad)),     # 夏至最大，冬至最小
        "jin": max(0, -math.cos(rad)),    # 秋分最大，春分最小
        "shui": max(0, -math.sin(rad)),   # 冬至最大，夏至最小
    }
```

### 2. 土引力算法（基于图7五带分布）

```python
def tu_gravity(color: HSL, intensity: float) -> HSL:
    """
    土引力：越靠近中轴(S越小)越不受力，越远离(S越大)收敛越强
    类比图7：赤道(中轴)=热带(土的中和区)，两极=寒带(极端)
    """
    if color.s < 0.01 or intensity <= 0:
        return color
    # 引力与距轴距离成正比（远离中轴 = 高S = 强引力拉回）
    pull = intensity * color.s
    return HSL(
        h=color.h,
        s=color.s * (1 - pull * 0.5),
        l=color.l + pull * 0.2 * (0.5 - color.l)
    ).clamp()
```

### 3. 日周期维度（基于图2晨昏线）

```python
class DiurnalCycle:
    """
    日周期：地球自转产生的昼夜交替
    对应八字中的"时柱"
    基于图2晨昏线分割昼夜半球
    """
    def __init__(self, hour: int):
        self.hour = hour  # 0-23
        
    def yin_yang_ratio(self) -> float:
        """返回阳的比例：正午=1.0, 子夜=0.0"""
        # 正午12时 = 太阳最高 = 最阳
        # 子夜0时 = 太阳最低 = 最阴
        return 0.5 + 0.5 * math.sin(math.radians((self.hour - 6) * 15))
```

## 五、总结

1. ✅ 黄道面四行分布 —— 与所有公转图完全一致
