## 一、设计原则

### 1.1 核心决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 色彩空间 | **HSL**（柱坐标） | 与球体理论天然同构：H=方位角，S=径向距离，L=极角 |
| 叠加范式 | **滤色片算子复合**（520元清） | 五行=变换函数，非颜色值；叠加=函数复合，非颜色混合 |
| 运算顺序 | **本命 → 大运 → 流年**（由内向外） | 本命是"底子"，大运是"十年滤镜"，流年是"当年氛围" |
| 柱内合成 | **Overlay混合**（司辰5/30定稿） | 干支层(100%) + 神煞层叠色(30-60%) + 藏干纹理(15-30%) |
| 柱间合成 | **Alpha混合 + 滤色片调制** | 四柱色不是平均混合，是按命局权重调制后叠加 |

### 1.2 为什么不选RGB/CMYK直接混合？

```
RGB加法（发光叠加）    → 只适合屏幕，越叠越亮，土会消失
CMYK减法（颜料调和）   → 越叠越暗，火会吞掉一切
Alpha混合（半透明层）  → 可行但太平淡，无法表达五行"改变质感"的特性

滤色片算子复合        → ✅ 每个五行是变换函数，不改变亮度总量，改变质感方向
```

## 二、球体坐标系定义

### 2.1 从HSL到球坐标

```
HSL圆柱坐标          球体坐标（球心=纯灰土）
┌─────────────┐      ┌─────────────────────────────┐
│ H: 0-360°   │  →   │ θ (azimuth): 五行方位角      │
│ S: 0-100%   │  →   │ r (radius):  饱和度距心距离   │
│ L: 0-100%   │  →   │ φ (polar):   明度极角        │
└─────────────┘      └─────────────────────────────┘

映射关系:
  θ = H（方位角 = 色相，但五行各占扇区而非连续环）
  r = S/100（径向 = 饱和度，0=球心纯灰，1=球面纯色）
  φ = π × (1 - L/100)（极角 = 明度，0=北极白，π=南极黑）
```

### 2.2 五行扇区（球面分布）

球面按日晷8字S线分割为五大扇区，非均匀分布：

```
                北极（白·阳之极）
                   φ=0
                    │
     木扇区          │         火扇区
   θ=60°~150°       │      θ=150°~240°
   （春·生发）       │      （夏·炎上）
                    │
   西────────────球心(灰·土)────────────东
   （金）            │                  （木）
                    │
     水扇区          │         金扇区
   θ=300°~60°    φ=π           θ=240°~300°
   （冬·润下）       │        （秋·从革）
                    │
                南极（黑·阴之极）
```

> **关键**: 五行扇区在球面上不是等分扇形，而是按十二长生螺旋的"势力范围"自然划分。木扇区宽90°（亥→卯→未），火扇区宽90°（寅→午→戌），依此类推。

### 2.3 十二长生螺旋线

每根天干对应一条从球面螺旋进入球心的S线：

```
天干S线方程（参数t从0到1，0=长生/球面，1=墓/球心）:
  θ(t) = θ_长生 + (θ_帝旺 - θ_长生) × sin(πt/2)
  r(t) = 1 - t^0.7          （饱和度衰减，非线性，帝旺时仍高饱和）
  φ(t) = φ_长生 + (φ_墓 - φ_长生) × t

关键节点:
  t=0   长生:  r≈1.0（球面，纯度高）
  t=0.3 帝旺:  r≈0.7（仍较纯，但开始入球）
  t=0.6 病/死: r≈0.4（明显发灰）
  t=1.0 墓:    r≈0.0（球心，纯灰）
```

> 指数0.7的选择：帝旺时颜色仍应鲜明（r≈0.7），衰退期快速入灰。

## 三、滤色片算子定义（五行变换函数）

### 3.1 算子签名

```python
def wuxing_operator(color: HSL, intensity: float) -> HSL:
    """
    输入: 当前HSL色 + 该五行作用强度(0.0~1.0)
    输出: 变换后的HSL色
    约束: 输出色仍在有效色域内(H∈[0,360), S∈[0,1], L∈[0,1])
    """
```

### 3.2 五大算子

#### 木算子 · 生发（让颜色"更活"）

```
木(color, i) → color'
  ΔH = +5° × i × (1-S)       # 低饱和时色相偏移更大（唤醒灰色）
  ΔS = +0.15 × i × sin(πL)   # 中等明度时增饱和最多（避开死黑/死白）
  ΔL = +0.03 × i × (0.5-L)   # 向中性明度靠拢（L<0.5提亮，L>0.5压暗）
```

> **直觉**: 木是"生机"，让沉闷的颜色鲜活，让过亮的颜色沉稳，向"生动"收敛。

#### 火算子 · 炎上（让颜色"更亮"）

```
火(color, i) → color'
  ΔH = -8° × i × S            # 高饱和时向暖偏（橙红方向）
  ΔS = +0.10 × i × L          # 高明度时增饱和（亮上加艳）
  ΔL = +0.12 × i × (1-L^0.5)  # 整体提亮，越暗提得越多（开灯效应）
```

> **直觉**: 火是"光明"，照亮暗处，让暖色更暖，但不改变冷色的色相本质。

#### 土算子 · 稼穑（让颜色"更稳"）

```
土(color, i) → color'
  ΔH = 0                       # 土不偏色，守中
  ΔS = -0.20 × i × S           # 直接降饱和（向灰色靠拢）
  ΔL = -0.05 × i × (L-0.5)     # 向中性明度靠拢（亮则压，暗则提）
```

> **直觉**: 土是"中和"，消弭极端，向球心（纯灰）收敛。这是唯一让颜色变"素"的算子。

#### 金算子 · 从革（让颜色"更冷"）

```
金(color, i) → color'
  ΔH = +10° × i × (1-S)       # 低饱和时向冷偏（蓝青方向）
  ΔS = -0.05 × i              # 微降饱和（金属光泽的克制感）
  ΔL = +0.08 × i × L          # 高明度时更亮（银白感）
```

> **直觉**: 金是"肃杀"，给颜色加一层冷冽的锋芒，但不彻底改色。

#### 水算子 · 润下（让颜色"更深"）

```
水(color, i) → color'
  ΔH = +5° × i × S            # 高饱和时向蓝偏
  ΔS = +0.08 × i × (1-L)      # 暗部增饱和（深海效应）
  ΔL = -0.15 × i × L^0.5      # 整体压暗，越亮压得越多（下沉效应）
```

> **直觉**: 水是"深邃"，让颜色沉下去，暗处更深邃，但不彻底变黑。

### 3.3 算子合成顺序

```
当多个五行同时作用于同一颜色时（如柱内叠加），按生克顺序复合：

复合顺序: 木 → 火 → 土 → 金 → 水（相生顺序，顺行叠加）

即: result = 水( 金( 土( 火( 木(color, i_木), i_火), i_土), i_金), i_水)

理由: 相生顺序符合"生发→炎上→中和→肃杀→润藏"的自然节律。
      若用相克顺序，会产生对抗抵消，不适合色彩调和。
```

## 四、三层球体叠加规则

### 4.1 三层定义

| 层 | 对应命理 | 变换性质 | 时间尺度 | 类比 |
|----|---------|---------|---------|------|
| **本命球体** B | 四柱八字原局 | 基准色场，不变 | 终身 | "肤色" |
| **大运球体** D | 十年大运 | 滤色片变换 T_D | 10年 | "十年滤镜" |
| **流年球体** L | 当年流年 | 滤色片变换 T_L | 1年 | "当年氛围" |

### 4.2 叠加公式

```
当前真实颜色 C = T_L( T_D( B ) )

其中:
  B = 四柱柱内叠加后的合并色（见4.3）
  T_D = 大运五行算子复合（大运干支对应的五行强度→算子）
  T_L = 流年五行算子复合（流年干支对应的五行强度→算子）
```

### 4.3 柱内叠加（司辰5/30规则的形式化）

以单柱为例（如月柱 = 己酉）：

```
Step 1: 干支层底色
  干色 = wuxing_color(天干五行, 干权重)
  支色 = wuxing_color(地支五行, 支权重)
  干支底色 = BlendMode_Overlay(干色, 支色)

Step 2: 神煞层叠色
  对每个显式神煞（按优先级排序）:
    煞色 = wuxing_color(神煞五行, 神煞权重)
    神煞层 = AlphaBlend(神煞层, 煞色, 神煞不透明度)
  神煞叠色 = 神煞层

Step 3: 藏干层纹理
  对每个显式藏干:
    藏色 = wuxing_color(藏干五行, 藏干权重)
    藏干纹理 = AlphaBlend(藏干纹理, 藏色, 藏干不透明度)

Step 4: 柱色合成
  柱色 = BlendMode_Overlay(BlendMode_Overlay(干支底色, 神煞叠色), 藏干纹理)
```

### 4.4 四柱合并（新增规则）

四柱色不是简单平均，而是按命局权重调制：

```
命局权重分配:
  日柱权重 = 0.40（日主为核心）
  月柱权重 = 0.25（月令提纲）
  时柱权重 = 0.20（行动输出）
  年柱权重 = 0.15（祖上根基）

合并方式:
  本命色 B = weighted_blend([年柱色, 月柱色, 日柱色, 时柱色],
                             [0.15, 0.25, 0.40, 0.20],
                             mode='cubic')  # 见5.2
```

> 为什么用cubic不用linear？四色加权平均会产生"灰色泥沼"（所有颜色中和成灰）。cubic混合保留主导色的鲜明度。

## 五、混合模式详述

### 5.1 BlendMode.Overlay（柱内叠加）

```python
def blend_overlay(base, blend):
    """
    标准Overlay混合（Photoshop公式）
    base: 底层色(RGB, 0~1)
    blend: 叠加层色(RGB, 0~1)
    """
    result = []
    for b, l in zip(base, blend):
        if b < 0.5:
            result.append(2 * b * l)
        else:
            result.append(1 - 2 * (1-b) * (1-l))
    return result
```

> 特点：暗部相乘（变暗），亮部滤色（变亮），保留对比度。

### 5.2 CubicWeightedBlend（四柱合并）

```python
def cubic_weighted_blend(colors, weights):
    """
    带权重的高次混合——避免灰色泥沼
    colors: [RGB1, RGB2, RGB3, RGB4]
    weights: [w1, w2, w3, w4], sum=1
    """
    # 先归一化权重为幂次
    powers = [w**0.7 for w in weights]  # 0.7次方：削弱小权重的稀释作用
    sum_powers = sum(powers)
    normalized = [p/sum_powers for p in powers]

    # 分量级加权（非简单线性）
    result = [0, 0, 0]
    for rgb, w in zip(colors, normalized):
        for i in range(3):
            result[i] += (rgb[i] ** 2) * w  # 平方加权：放大主导色差距

    # 还原
    result = [math.sqrt(c) for c in result]
    return clamp(result)
```

> 原理：平方加权让高值更高、低值更低，保留主色的鲜明度，避免四个颜色平均成灰。

## 六、从命理数据到算子的映射

### 6.1 输入：司辰JSON

```json
{
  "wuxing_weights": {"jin":0.15, "mu":0.10, "shui":0.20, "huo":0.35, "tu":0.20},
  "yong_shen": ["甲木", "庚金"],
  "da_yun": {"gan":"癸", "zhi":"丑", "wuxing":{"shui":0.6, "tu":0.4}},
  "liu_nian": {"gan":"乙", "zhi":"巳", "wuxing":{"mu":0.4, "huo":0.6}}
}
```

### 6.2 映射流程

```
wuxing_weights → 本命五行算子强度
  jin:0.15 → 金算子 intensity=0.15
  mu:0.10 → 木算子 intensity=0.10
  ...

da_yun.wuxing → 大运五行算子强度
  shui:0.6 → 水算子 intensity=0.6
  tu:0.4 → 土算子 intensity=0.4

liu_nian.wuxing → 流年五行算子强度
  mu:0.4 → 木算子 intensity=0.4
  huo:0.6 → 火算子 intensity=0.6
```

> 注意：强度值就是算子的i参数，直接传入。

## 七、验证与自检

### 7.1 数学自检项

| # | 检查项 | 通过标准 |
|----|--------|---------|
| 1 | 算子封闭性 | 任意HSL输入经算子变换后仍在[0,360)×[0,1]×[0,1]内 |
| 2 | 土算子幂等性 | 土(土(color,i1),i2) ≈ 土(color, i1+i2)（近似，允许5%误差） |
| 3 | 五行和约束 | 五行强度之和∈[0.8, 1.2]（允许命理计算的合理误差） |
| 4 | 叠加单调性 | 强度i增大时，变换效果方向一致（不反向） |
| 5 | 中性色稳定性 | 纯灰(H任意,S=0,L=0.5)经任意算子后仍≈灰（L变化<0.1） |

### 7.2 命理一致性检查

| # | 检查项 | 通过标准 |
|----|--------|---------|
| 6 | 用神色保护 | yong_shen对应的五行算子强度 ≥ 0.15（不被完全压制） |
| 7 | 忌神色弱化 | ji_shen对应的五行算子强度 ≤ 0.35（不主导） |
| 8 | 大运方向一致 | 大运算子变换方向与用神同向（助用神则加，克用神则减） |
