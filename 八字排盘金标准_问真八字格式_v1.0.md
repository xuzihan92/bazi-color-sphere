## 六、三人共用校验流程

### 洞微（盲派）

排盘后逐项核对：
1. **日柱必须是丁未** → 若算成丙午，引擎误差，直接修正
2. **时柱必须是丙午** → 日主丁，午时应为丙午（非甲午）
3. **大运第4运必须是癸丑** → 男命阳年顺排
4. 盲派技法（串宫压运、十二神煞）基于此盘分析

### 司辰（色彩映射）

1. **无需自己排盘**，直接采用以下四柱输入引擎：
   - 年柱：壬申 → H=270° S=0.70 L=0.20 T=5300K
   - 月柱：己酉 → H=null S=0.50 L=0.75 T=6500K
   - 日柱：**丁未** → H=8° S=0.50 L=0.55 T=3600K
   - 时柱：丙午 → H=0° S=0.85 L=0.50 T=2600K
2. **若日柱误用丙午** → 引擎会输出错误颜色（H=0°纯红），必须修正为丁未

### 甄天（八字分析）

1. **日主必须是丁火** → 若误用丙火，格局判断全错
2. **偏财格** → 月令酉金，非正印格/伤官格
3. **用神木火** → 身弱喜印比，非从财格
4. 所有神煞、十神定位以本盘藏干为准
