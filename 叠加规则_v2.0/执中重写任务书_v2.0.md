## 一、架构变更总览

```
v1.0（5/31，已废弃）            v2.0（6/6，新架构）
─────────────────────────────────────────────────────
五个独立算子                     四个算子 + 土作为边界条件
土=H=45°独立色相点              土=明度轴（L轴），从0(黑)到1(白)
七彩球面拼图                     黄道面四行 + 中轴土
```

## 二、需修改的 6 个核心项

### 2.1 土的算子定义 → 删除，改为轴收敛

**v1.0 错误** (`WuxingOperator.tu`, 第77-89行):
```python
def tu(color, intensity):
    # 土不偏色，降饱和，向中性明度靠拢
    return HSL(
        h=color.h,  # 土不偏色——但这意味着土和其他四行同级别
        s=color.s - 0.20 * i * color.s,
        l=color.l - 0.05 * i * (color.l - 0.5)
    )
```

**v2.0 正确做法**: `tu` 不是独立算子。它是四行算子的**极值约束**：
```python
# 土=明度轴收敛：当任意颜色经过四行变换后，
# 根据其距轴距离(S)自动向L轴收束
def _tu_convergence(color: HSL, intensity: float) -> HSL:
    """土引力：离轴越远(S越大)，强度越高时收敛越强"""
    if intensity <= 0:
        return color
    convergence = intensity * (1 - color.s)  # S=0(轴上)不受影响
    return HSL(
        h=color.h,
        s=color.s * (1 - convergence * 0.3),
        l=color.l + convergence * 0.1 * (0.5 - color.l)
    ).clamp()
```

### 2.2 算子数量 → 5→4

**v1.0**: `order = ["mu", "huo", "tu", "jin", "shui"]`

**v2.0**: `order = ["mu", "huo", "jin", "shui"]`（土不在复合序列中）

土运算符只在四种情况下触发：
1. 当任一算子强度 ≥ 0.8（极值保护）
2. 当四行算子全部施加后，作为**后处理**统一收敛
3. 当中间色 S > 0.9（纯色保护——防止颜色过于鲜艳脱离土球）
4. 当最终色需要体现"土的中和之力"时

关键变化:

- `木` 从 105° → 120°（纯绿）
- `金` 从 285° → 210°（从冷白到肃杀白）
- `水` 从 345° → 270°（从紫蓝到深海蓝）
- `土` 从 45° → **不存在**（土不是球面上的色相点）

### 2.4 HSL 坐标语义 → 重新定义

```
v1.0:                           v2.0:
H = 五行扇区(5个等分?)          H = 黄道面方位角（春0°/夏90°/秋180°/冬270°）
S = 距球心距离                   S = 距中轴距离（土引力：S→0则回到土）
L = 球面高度                     L = 中轴位置（土的本色：0=南极黑，0.5=球心灰，1=北极白）
```

### 2.5 算子复合顺序 → 从"串行相生"改为"黄道顺行+轴收敛"

**v1.0**: 木→火→土→金→水（固定顺序串行）

**v2.0**: 
```
Step 1: 四行算子按黄道顺行施加（春→夏→秋→冬）
Step 2: 土引力后处理——根据色点到中轴的距离(S)施加轴收敛
Step 3: 如果有极值（L≈0或L≈1），额外触发极值约束
```

### 2.6 土算子语义 → `gravity` 而非 `transform`

```python
# 旧：土是主动变换
tu(color, i) → 降饱和、向灰偏

# 新：土是引力场
class TuGravity:
    """
    土引力 = 使颜色向明度轴收敛的力
    - 离轴越远(S越大) → 引力越强
    - 强度越大 → 收敛越快
    - 在轴上(S=0) → 不受影响（已经是纯土色）
    """
    def apply(self, color: HSL, intensity: float) -> HSL:
        if color.s < 0.01 or intensity <= 0:
            return color
        pull = intensity * color.s  # 引力 = 强度 × 距轴距离
        return HSL(
            h=color.h,  # 不改变色相
            s=color.s * (1 - pull * 0.5),
            l=color.l + pull * 0.2 * (0.5 - color.l)  # 向球心(L=0.5)靠拢
        ).clamp()
```
