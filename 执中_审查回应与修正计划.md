## 二、逐项回应

### P0-1：四行色相角映射不一致（元清 🟡）

**问题**: 黄道面方位角（春分0°/夏至90°/秋分180°/冬至270°）与 HSL 色相（木120°/火0°/金210°/水270°）之间偏移量不一致。

**根因**: 四种颜色在光谱中的位置天然不均匀——木(绿)和火(红)在色相环上相差120°，但天文上春分→夏至是90°。

**修正方案（采纳元清建议B）**：

不用固定色相角，改用**色温映射**：

```python
# 色温（Kelvin）对应季节天光
COLOR_TEMP = {
    "春分": 5500,   # 中性白光（类似正午阳光）
    "夏至": 3000,   # 暖黄光（日出/日落色温）
    "秋分": 6500,   # 冷白光（阴天色温）
    "冬至": 8000,   # 蓝白光（阴影/雪地色温）
}

def temp_to_hsl(temp_k: float) -> tuple:
    """色温(K) -> HSL色相近似映射"""
    # 简化模型：
    # 3000K = 暖红/橙 (H≈30°)
    # 5500K = 中性白 (H≈120° 绿/或0°红取决于实现)
    # 8000K = 冷蓝 (H≈220°)
    # 实际应使用标准色温->RGB转换（如McCamy公式）
    if temp_k < 4000:
        h = 30 + (temp_k - 3000) / 1000 * 90   # 30° -> 120°
    elif temp_k < 6000:
        h = 120 + (temp_k - 4000) / 2000 * 60  # 120° -> 180°
    else:
        h = 180 + (temp_k - 6000) / 2000 * 120 # 180° -> 300°
    return h % 360
```

**短期回退**: 如果色温映射太复杂，v2.0 第一版仍用固定色相角，但**明确标注"非线性映射，待色温模型替换"**。

### P0-3：缺少四柱→球面坐标映射规范（衍微 🔴）

**问题**: 引擎接口只定义了输入输出格式，但没有定义「壬申→球面上哪个点」。

**修正方案（立即补写）**：

```python
class BaziToSphereMapper:
    """四柱 -> 球面坐标映射 v1.0（简化版）"""
    
    def map_pillar(self, gan: str, zhi: str) -> tuple:
        """
        单柱 -> 球面坐标 (r, theta, phi)
        
        简化策略：
        - 天干定五行 -> 黄道面方位角(phi)
        - 地支定十二长生 -> 中轴倾角(theta)
        - 柱的重要性（年/月/日/时）-> 半径(r)
        """
        # 天干五行映射
        gan_wuxing = {
            "甲": "木", "乙": "木", "丙": "火", "丁": "火",
            "戊": "土", "己": "土", "庚": "金", "辛": "金",
            "壬": "水", "癸": "水"
        }
        
        # 地支藏干（取主气）
        zhi_canggan = {
            "子": "癸", "丑": "己", "寅": "甲", "卯": "乙",
            "辰": "戊", "巳": "丙", "午": "丁", "未": "己",
            "申": "庚", "酉": "辛", "戌": "戊", "亥": "壬"
        }
        
        # 五行 -> 黄道面方位角
        wuxing_phi = {"木": 0, "火": 90, "金": 180, "水": 270, "土": 45}
        
        wx = gan_wuxing.get(gan, "土")
        phi = np.radians(wuxing_phi.get(wx, 45))
        
        # 地支 -> 十二长生位置（决定中轴倾角）
        # 简化：假设日主在"帝旺"位时 theta=0（赤道），在"墓"位时 theta=45°
        # 实际应根据六十甲子查十二长生表
        theta = np.radians(30)  # 默认30°偏轴
        
        # 半径：由柱的重要性决定（日柱最靠近球面）
        r = 0.8  # 默认
        
        return (r, theta, phi)
    
    def map_birth(self, year, month, day, hour) -> tuple:
        """
        四柱合成 -> 本命球坐标
        
        简化版：四柱加权平均
        权重：日柱(0.4) > 月柱(0.25) > 时柱(0.2) > 年柱(0.15)
        """
        pillars = [year, month, day, hour]
        weights = [0.15, 0.25, 0.4, 0.2]
        
        coords = [self.map_pillar(p["gan"], p["zhi"]) for p in pillars]
        
        # 加权平均（球坐标需转直角坐标再平均）
        x = sum(w * r * np.sin(t) * np.cos(p) for w, (r, t, p) in zip(weights, coords))
        y = sum(w * r * np.sin(t) * np.sin(p) for w, (r, t, p) in zip(weights, coords))
        z = sum(w * r * np.cos(t) for w, (r, t, p) in zip(weights, coords))
        
        # 转回球坐标
        r_avg = np.sqrt(x**2 + y**2 + z**2)
        theta_avg = np.arccos(z / r_avg)
        phi_avg = np.arctan2(y, x)
        
        return (r_avg, theta_avg, phi_avg)
```

**说明**: 这是v1.0简化映射（查表法），后续由司辰提供更精确的「六十甲子→十二长生→球面坐标」映射表。

### P1-3：整合 520 元清「五行=滤色片」算子层（衍微 🟡）

**状态**: 同意。v3.0 几何层（坐标）和算子层（变换）是互补的。

**实现计划**:

```python
class WuxingFilter:
    """五行滤色片——颜色变换算子（520元清模型）"""
    
    def mu(self, color: HSL) -> HSL:
        """木: 生发 (+green shift, +saturation)"""
        return HSL((color.h + 15) % 360, min(1, color.s * 1.2), color.l)
    
    def huo(self, color: HSL) -> HSL:
        """火: 炎上 (+L, warm hue)"""
        return HSL((color.h - 10) % 360, color.s, min(1, color.l * 1.15))
    
    def tu(self, color: HSL) -> HSL:
        """土: 稼穑 (desaturate, -> gray)"""
        return HSL(color.h, color.s * 0.7, color.l + 0.05 * (0.5 - color.l))
    
    def jin(self, color: HSL) -> HSL:
        """金: 从革 (-L, cool hue)"""
        return HSL((color.h + 20) % 360, color.s * 0.9, color.l * 0.95)
    
    def shui(self, color: HSL) -> HSL:
        """水: 润下 (-L, -saturation)"""
        return HSL((color.h + 10) % 360, color.s * 0.85, color.l * 0.9)
    
    def chain(self, color: HSL, filters: list[str]) -> HSL:
        """滤镜串联: e.g., ["mu", "huo"] = 木生火"""
        result = color
        for f in filters:
            result = getattr(self, f)(result)
        return result
```

**与几何层的配合**:
- 几何层决定「基础色在哪」（坐标）
- 算子层决定「颜色怎么变」（滤镜）
- 最终色 = 几何坐标色 + 算子变换效果

def k_segmented(t, t_peak=0.5, sigma=0.15, k_base=0.3, delta_k=0.4):
    """
    分段螺距：帝旺附近变化快，墓绝附近变化慢
