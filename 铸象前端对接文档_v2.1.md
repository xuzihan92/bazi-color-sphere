## 二、main_case 字段说明

### 2.1 本命色

```json
"birth_color": {
  "hex": "#CF495F",    // 直接用于CSS背景色
  "H": 350.2,           // 色相角 0-360°
  "S": 0.583,           // 饱和度 0-1
  "L": 0.55,            // 明度 0-1
  "T": 4575             // 色温(K)，用于描述文字
}
```

**前端用法**:
- `hex` → 色卡背景、主色调展示
- `H/S/L` → 悬停tooltip显示技术参数
- `T` → "4575K 偏暖" 这样的文字描述

### 2.2 四柱

```json
"pillars": {
  "year": {
    "ganzhi": "壬申",    // 干支名
    "gan": "壬",         // 天干
    "zhi": "申",         // 地支
    "wuxing": "水",      // 五行（用于图标/标签色）
    "H": 270,            // 色相（土为null）
    "S": 0.7,
    "L": 0.2,
    "T": 5300,
    "hex": "#320F56",    // 柱色（直接显示）
    "stage": "长生",      // 十二长生
    "wangshuai": "相"    // 旺衰
  },
  "month": { ... },
  "day": { ... },
  "hour": { ... }
}
```

**前端用法**:
- 四柱卡片：每个柱一个色块，`hex`做背景
- 天干/地支标签：`wuxing`决定标签颜色（木=绿、火=红、土=黄、金=白/灰、水=蓝）
- 悬停详情：显示 H/S/L/T + stage + wangshuai
- **土柱特殊处理**：`H=null` 时，色块显示为灰白渐变（不显示色相环角度）

### 2.3 五行平衡

```json
"wuxing_balance": {
  "木": 0.024,
  "火": 0.349,
  "土": 0.303,
  "金": 0.236,
  "水": 0.088
}
```

**前端用法**:
- 饼图/环形图：5个扇形，按数值比例
- 颜色映射：木=#4CAF50, 火=#F44336, 土=#FFC107, 金=#9E9E9E, 水=#2196F3
- 当前高亮：最大值（火34.9%）高亮或放大

### 2.4 大运列表

```json
"da_yun": [
  {
    "ganzhi": "庚戌",
    "period": "第1运",
    "intensity": 0.2,      // 大运影响强度
    "H": 339.0,
    "S": 0.636,
    "L": 0.55,
    "T": 5100,
    "hex": "#D54376"       // 大运叠加色
  },
  ...
]
```

**前端用法**:
- 大运时间轴：10个点，可点击切换
- 当前大运高亮：`current_da_yun` 字段标记当前是哪个
- 切换效果：点击大运 → 主色块渐变到该运hex

## 三、交互功能实现（B路线）

### 3.1 大运切换

```javascript
// 伪代码
data.main_case.da_yun.forEach(dy => {
  // 渲染大运按钮
  button.style.background = dy.hex;
  button.textContent = dy.ganzhi;
  
  button.onclick = () => {
    // 切换主色显示
    mainColorBlock.style.background = dy.hex;
    // 显示参数
    showParams(dy.H, dy.S, dy.L, dy.T);
  };
});
```

### 3.2 八字对比

```javascript
// compare_cases 里有3个典型命例
data.compare_cases.forEach(c => {
  // 渲染对比卡片
  card.style.background = c.birth_color.hex;
  // 显示五行饼图
  renderPie(c.wuxing_balance);
});
```

### 3.3 色温/季节筛选

**前端过滤，不动引擎**:
```javascript
// 按色温范围过滤大运
const warmDaYun = data.main_case.da_yun.filter(dy => dy.T < 4000);  // 暖运
const coolDaYun = data.main_case.da_yun.filter(dy => dy.T > 5000);  // 冷运
```

## 五、视觉建议（给琳琅参考）

1. **主色块**：用 `birth_color.hex` 做大背景或中心色卡
2. **四柱排列**：从左到右 年→月→日→时，日柱放大突出
3. **土柱处理**：`H=null` 的柱显示为灰白渐变，不加色相环标记
4. **大运时间轴**：水平排列10个点，当前运发光/脉冲效果
5. **色温文字**：T>5000K 显示"偏冷"，T<4000K 显示"偏暖"
