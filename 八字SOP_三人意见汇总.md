### 4.2 SOP v1.1 核心改进（已更新）

1. **盲算协议三硬约束**写入 SOP
   - 输入限制：仅出生时间 + 性别
   - 屏蔽信息：姓名、职业、健康、感情、近期事件
   - 锁定机制：盲算完成后先锁结果，才能看 A 的报告

2. **分歧触发扩圈**替代"可选增强"
   - A/B 一致 → 跳过 Phase 3
   - 五行权重差 > 0.08 → 触发璇玑星盘对照
   - 同一事件吉凶相反 → 触发钧玄多流派分析

3. **doubt_flags 字段**加入 JSON 输出
   - 字段名、争议原因、影响分析、建议验证方向
   - 下游扫一眼就知道该盯哪里

4. **共识层输出接口**
   - 给采蓝/衍微的输入从单一 `wuxing_weights` 升级为「共识+异议」结构
   - 下游知道主流判断是什么，也知道哪里可能有偏差

5. **长期 alerts**
   - 先天缺木等长期指标写入一次，标记为"长期"
   - 颐恒/度衡可订阅，不需要每天重复报
