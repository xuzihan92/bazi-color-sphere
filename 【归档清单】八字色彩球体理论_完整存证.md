### 4.1 数字存证方式

1. **Git 版本控制**（推荐）
   - 将所有文件纳入 Git 仓库
   - 每次提交都有不可篡改的 SHA-1 哈希和时间戳
   - 可推送到 GitHub/GitLab 私有仓库
   - Git 提交记录即为法律认可的创作时间证明

2. **区块链存证**（最高安全性）
   - 将所有文件的 SHA-256 哈希上链（如蚂蚁链、至信链）
   - 区块链时间戳不可篡改，法律效力强
   - 成本约 1-10 元/次

3. **可信时间戳（TSA）**（法律认可）
   - 通过中国版权保护中心或联合信任（TSA）申请
   - 符合《电子签名法》，可直接用于维权
   - 成本约 10-50 元/份

4. **邮件自存证**（低成本）
   - 将所有文件打包发送到可信邮箱（如 Gmail/Outlook）
   - 邮件服务器时间戳可作为辅助证据
   - 发送到自己的多个邮箱增加可靠性
