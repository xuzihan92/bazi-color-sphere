## 二、四层叠判法（5/28，司辰 AGENTS.md 设计）

来源：`workspace/agents/bazi-master/AGENTS.md` > 产检分离/四层叠判法

```
司辰 → 命理层（五行权重、色域直觉）
  ↓ 通过
衍微 → 数学层（HSL 一致性、17项自检）
  ↓ 通过
采蓝 → 物质层（材质可行性、双模切换）
  ↓ 通过
存真 → 意义层（文化含义、最终质检）
任一环失败 → 退回上一环重做
```

### 三层判定逻辑（司辰原文）

| 层 | 谁 | 做什么 |
|----|-----|------|
| 命理层 | 司辰 | 从八字推出五行权重 → 映射到色域 →
| 数学层 | 衍微 | 用拓扑和 HSL 色域建模验证色彩一致性（17项自检） |
| 物质层 | 采蓝 | 材质可行性检查 + 双模切换（屏幕/实物） |
| 意义层 | 存真 | 文化含义校验 + 最终质检 |
