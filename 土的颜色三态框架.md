# 土的颜色三态框架

> 作者: 司辰 | 日期: 2026-06-08 | 基于梓涵讨论  
> 关联: 八字色彩球体模型 v3.0

---

## 传统理解的问题

传统五行色彩说「土＝黄」，这在二维色轮上是合理的——木绿火红土黄金白水黑，五个点各占一位。

但球体模型里这说不通。土在五行中不参与四行环流，它是另外四行运行的**场**、**承载**、**平台**。球体模型给了土一个更好的位置：**轴心**。

---

## 土的三态

```
        极昼白（火生土）
       /    ↕ 土在极位    \
      /                     \
   木(绿)                  金(白蓝)
  (春分)                  (秋分)
      \                     /
       \    ↕ 土在赤道    /
        火(红)──────水(黑)
        (夏至)      (冬至)
              ↕ 土在核
            轴心承载
              ↑
         极夜黑（土克水）
```

### 态一：轴心 — 承载万有

- **位置**：球体正中心
- **颜色**：无固有色——它不发光，它被光照
- **五行角色**：戊己土为枢纽，不参与环面，但所有四行的颜色都投射在它身上
- **类比**：地球的地核——你看不到它，但一切都在它之上发生

### 态二：赤道过渡 — 四季土

- **位置**：球面黄道带上，四行之间的过渡区
- **颜色**：随两侧元素渐变，非单一黄色

| 地支 | 过渡 | 颜色 | 感觉 |
|------|------|------|------|
| 辰 | 春→夏（木→火） | 黄绿→暖褐 | 嫩芽转繁茂 |
| 未 | 夏→秋（火→金） | 橙褐→灰金 | 熟透转收敛 |
| 戌 | 秋→冬（金→水） | 灰白→暗蓝灰 | 肃杀转潜藏 |
| 丑 | 冬→春（水→木） | 暗黑→青灰 | 冰封转萌动 |

土在此不是一种颜色——是四种过渡色的统称。

### 态三：两极 — 阴阳翻面点

- **位置**：球体南北极
- **极昼（北）**：火旺极而生土 → 最亮最白。火不再燃烧，转为承载万物的温热大地
- **极夜（南）**：水旺极而土克水 → 最暗最黑。水不再流动，是冻土之下的潜渊
- **五行含义**：土在两端不是「中间色」，而是**阴阳转换的支点**

---

## 与金水黑白的关系

传统上「金=白、水=黑」，但在球体模型里，黑白首先是土的两极表现：

- **北极白**：火极生土的白光 → 金的金气在此被承载
- **南极黑**：水极土克的深渊 → 水的寒性在此被约束

金水只是被土承载着、在两极呈现出黑白。所以「黑白既是水也是金也是土」——这不矛盾，是层次不同：
- 金水解释「显什么色」
- 土解释「这颜色在哪发生的、为什么是这个极端」

---

## 对引擎的影响

- 土五行（戊己天干）的 H 值设为 null——不参与环面计算
- 土柱对颜色的影响通过 L 轴（十二长生位置）和 S 轴（旺衰）体现
- 辰戌丑未四库独立标注过渡色温，不归入单一行
- 极昼/极夜位置作为边界条件，L=0（南极黑）和 L=1（北极白）需要引擎特殊处理
