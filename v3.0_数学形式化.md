## 一、坐标系定义

### 1.1 标准直角坐标系

以球心O为原点，建立右手直角坐标系：
- **X轴**: 指向春分点（震3/东方）
- **Y轴**: 指向夏至点（离9/南方）
- **Z轴**: 指向北极（垂直于黄道面向上）

### 1.2 中轴（巽4→中5→乾6）的精确方向

根据后天八卦方位和v3.0拓扑：

| 点 | 方位 | 球坐标(θ, φ) | 直角坐标(x, y, z) |
|----|------|--------------|-------------------|
| 巽4 | 东南偏上 | θ=45°, φ=45° | (√2/2·sin45°, √2/2·sin45°, cos45°) ≈ (0.5, 0.5, 0.707) |
| 中5 | 球心 | — | (0, 0, 0) |
| 乾6 | 西北偏下 | θ=135°, φ=45° | (-0.5, -0.5, -0.707) |

**中轴方向向量**: **d** = 乾6 - 巽4 = (-1, -1, -1.414)

归一化：
```
d̂ = (-1, -1, -1.414) / √(1 + 1 + 2) = (-1, -1, -1.414) / 2 = (-0.5, -0.5, -0.707)
```

**中轴倾斜角**: α = arccos(|d̂ · ẑ|) = arccos(0.707) ≈ **45°**

> 注：此处45°是简化模型。真实黄赤交角为23.5°，后续可校准。

### 1.3 球坐标系（以中轴为极轴）

建立以中轴为Z'轴的球坐标系 (r, θ', φ')：
- r ∈ [0, R]: 到球心距离（R=1为单位球）
- θ' ∈ [0, π]: 与中轴的夹角（极角）
- φ' ∈ [0, 2π): 绕中轴的方位角（方位角）

坐标变换矩阵：
```
将标准坐标(x,y,z)转换到中轴坐标系(x',y',z')：

z' = x·(-0.5) + y·(-0.5) + z·(-0.707)   # 沿中轴投影
x' = x·(0.707) + y·(-0.707)             # 垂直于中轴平面（简化）
y' = ...                                  # 需完整构造正交基
```

## 二、双螺旋参数方程

### 2.1 阳螺旋（上升链）

**路径**: 坎1 → 坤2 → 震3 → 巽4 → 中5

**参数方程**（从中轴坐标系描述）：

```python
import numpy as np

def yang_spiral(t, n=1.0, k=0.5):
    """
    阳螺旋参数方程
    
    Args:
        t: 参数 ∈ [0, 1]，t=0在坎1(球面)，t=1在中5(球心)
        n: 缠绕圈数（默认1圈=1年）
        k: 螺距系数（控制螺旋松紧）
    
    Returns:
        (r, theta_prime, phi_prime): 中轴球坐标
    """
    r = 1 - t                    # 从球面(r=1)到球心(r=0)
    phi_prime = 2 * np.pi * n * t  # 方位角随t线性增长
    theta_prime = np.pi/2 * (1 - k * t)  # 极角从90°向中轴收束
    
    return r, theta_prime, phi_prime
```

**关键点验证**：
- t=0 (坎1): r=1, θ'=90°, φ'=0° → 球面上，位于中轴赤道平面
- t=0.25 (坤2): r=0.75, θ'=82.5°, φ'=90° → 向中轴靠拢
- t=0.5 (震3): r=0.5, θ'=75°, φ'=180° → 中轴坐标系赤道对面
- t=0.75 (巽4): r=0.25, θ'=67.5°, φ'=270° → 靠近中轴
- t=1 (中5): r=0, θ'=45°, φ'=360° → 球心（所有角度汇聚）

### 2.2 阴螺旋（下降链）

**路径**: 中5 → 乾6 → 兑7 → 艮8 → 离9 → 坎1

**参数方程**：

```python
def yin_spiral(t, n=1.0, k=0.5):
    """
    阴螺旋参数方程
    
    Args:
        t: 参数 ∈ [0, 1]，t=0在中5(球心)，t=1在坎1(球面)
        n: 缠绕圈数
        k: 螺距系数
    
    Returns:
        (r, theta_prime, phi_prime): 中轴球坐标
    """
    r = t                        # 从球心(r=0)到球面(r=1)
    phi_prime = 2 * np.pi * n * t + np.pi  # 方位角反向+180°偏移
    theta_prime = np.pi/2 * (1 + k * t)    # 极角从90°向反方向收束
    
    return r, theta_prime, phi_prime
```

**关键点验证**：
- t=0 (中5): r=0 → 球心
- t=0.2 (乾6): r=0.2, φ'=36°+180°=216° → 中轴下端附近
- t=0.4 (兑7): r=0.4, φ'=72°+180°=252° → 向赤道下降
- t=0.6 (艮8): r=0.6, φ'=108°+180°=288° → 继续下降
- t=0.8 (离9): r=0.8, φ'=144°+180°=324° → 靠近北极
- t=1 (坎1): r=1, φ'=180°+180°=360°=0° → 回到坎1，与阳螺旋闭合

### 2.3 双螺旋的∞形状证明

**命题**: 阳螺旋和阴螺旋在三维空间中的并集呈∞（沙漏）形状。

**证明思路**:

1. 阳螺旋从球面(r=1)向球心(r=0)收束，同时绕中轴旋转n圈
2. 阴螺旋从球心(r=0)向球面(r=1)发散，同时绕中轴反向旋转n圈
3. 两条螺旋在中轴两侧对称分布（θ'互补）
4. 在垂直于中轴的任意截面上，两条螺旋形成两个相反的弧线
5. 这些弧线在投影中连接成S形，整体构成∞

**数学表达**：

取垂直于中轴的截面（z'=常数），两条螺旋的交点满足：

```
阳螺旋: r_yang = 1 - t_yang, φ_yang = 2πn·t_yang
阴螺旋: r_yin = t_yin, φ_yin = 2πn·t_yin + π

令 r_yang = r_yin = r（同一截面同一半径）:
1 - t_yang = t_yin = r

方位角差:
Δφ = φ_yin - φ_yang = 2πn·t_yin + π - 2πn·t_yang
    = 2πn·r + π - 2πn·(1-r)
    = 2πn·(2r - 1) + π

当 r = 0.5（赤道截面）:
Δφ = π

即两条螺旋在赤道截面上 diametrically opposite（直径两端）——形成∞的交汇点。
```

## 三、阴阳消长动力学

### 3.1 时间参数化

设时间参数 τ ∈ [0, 1]，τ=0为冬至，τ=0.25为春分，τ=0.5为夏至，τ=0.75为秋分，τ=1回到冬至。

### 3.2 阴阳强度函数

```python
import numpy as np

def yang_strength(tau):
    """阳螺旋强度 = sin(2π·tau)"""
    return np.sin(2 * np.pi * tau)

def yin_strength(tau):
    """阴螺旋强度 = cos(2π·tau)"""
    return np.cos(2 * np.pi * tau)

# 验证：
# tau=0（冬至）: yang=0（最短），yin=1（最长）
# tau=0.25（春分）: yang=1，yin=0（理论上应为等长，需调整）
# tau=0.5（夏至）: yang=0，yin=-1（绝对值最大）
```

**修正**: 更精确的表达应使用相位偏移：

```python
def yang_strength(tau):
    """
    阳螺旋强度
    冬至=0，春分=√2/2，夏至=1，秋分=√2/2
    """
    return np.sin(np.pi * tau)  # 使用半周期，tau∈[0,1]对应0到π

def yin_strength(tau):
    """
    阴螺旋强度
    冬至=1，春分=√2/2，夏至=0，秋分=√2/2
    """
    return np.cos(np.pi * tau)

# 验证守恒定律：
# yang² + yin² = sin²(π·tau) + cos²(π·tau) = 1 ✅
```

### 3.3 螺旋相位函数

球面上任意点P的螺旋相位由其在双螺旋上的位置决定：

```python
def spiral_phase(P, tau):
    """
    计算点P在当前时间tau的螺旋相位
    
    Args:
        P: (x, y, z) - 球面上点的直角坐标
        tau: 时间参数 ∈ [0,1]
    
    Returns:
        phase ∈ [-1, 1]: 正值=阳螺旋主导，负值=阴螺旋主导
    """
    # 计算P到中轴的距离
    d_axis = distance_to_axis(P)
    
    # 计算P在中轴坐标系中的方位角
    phi_prime = azimuth_angle(P)
    
    # 螺旋相位 = 阳强度·cos(φ') + 阴强度·sin(φ')
    y = yang_strength(tau)
    yi = yin_strength(tau)
    
    phase = y * np.cos(phi_prime) + yi * np.sin(phi_prime)
    
    # 根据到中轴距离加权（越靠近中轴，相位越显著）
    phase *= (1 - d_axis)
    
    return phase
```

### 4.2 动态色彩偏移（螺旋相位）

```python
def spiral_color_offset(P, tau, amplitude=0.3):
    """
    计算螺旋相位引起的色彩偏移
    
    Args:
        P: (x, y, z) - 球面上点
        tau: 时间参数
        amplitude: 偏移振幅（控制动态变化的强度）
    
    Returns:
        (dR, dG, dB) - RGB偏移量
    """
    phase = spiral_phase(P, tau)
    
    # 根据phase计算色相偏移
    # phase>0: 向阳偏（暖色增强）
    # phase<0: 向阴偏（冷色增强）
    
    hue_shift = phase * 30  # 最大±30°色相偏移
    
    # 饱和度偏移：phase绝对值越大，饱和度越高（色彩更鲜艳）
    sat_shift = abs(phase) * 0.2
    
    # 明度偏移：阳相增亮，阴相增暗
    lum_shift = phase * 0.1
    
    return hue_shift, sat_shift, lum_shift
```

```python
def get_color_4pillars(P, year_tau, month_tau, day_tau, hour_tau,
                       weights=None):
    """
    八字四柱多层螺旋色彩
    
    Args:
        P: 球面上点
        year_tau: 年柱时间相位
        month_tau: 月柱时间相位
        day_tau: 日柱时间相位
        hour_tau: 时柱时间相位
        weights: 四柱权重，默认[0.4, 0.3, 0.2, 0.1]
    
    Returns:
        (R, G, B) - 最终色彩
    """
    if weights is None:
        weights = [0.4, 0.3, 0.2, 0.1]
    
    # 各层螺旋偏移
    offsets = [
        spiral_color_offset(P, year_tau, amplitude=0.3),   # 年：主导
        spiral_color_offset(P, month_tau, amplitude=0.2),  # 月：次级
        spiral_color_offset(P, day_tau, amplitude=0.15),   # 日：微调
        spiral_color_offset(P, hour_tau, amplitude=0.1),   # 时：微调
    ]
    
    # 加权合成
    dH = sum(w * off[0] for w, off in zip(weights, offsets))
    dS = sum(w * off[1] for w, off in zip(weights, offsets))
    dL = sum(w * off[2] for w, off in zip(weights, offsets))
    
    # 基础色 + 合成偏移
    H_base, S_base, L_base = get_base_hsl(P)
    H = (H_base + dH) % 360
    S = np.clip(S_base + dS, 0, 1)
    L = np.clip(L_base + dL, 0, 1)
    
    return hsl_to_rgb(H, S, L)
```

```python
"""
八字色彩 · 后天八卦双螺旋球体模型 v3.0
数学形式化实现
"""

import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

class HoutianBaguaSpiralMath:
    """后天八卦双螺旋数学模型"""
    
    def __init__(self, axis_tilt=45.0, n_turns=1.0, k_pitch=0.5):
        self.axis_tilt = np.radians(axis_tilt)
        self.n_turns = n_turns
        self.k_pitch = k_pitch
        
        # 中轴方向向量（归一化）
        # 简化：中轴在XZ平面内倾斜
        self.axis_dir = np.array([
            np.sin(self.axis_tilt),
            0,
            np.cos(self.axis_tilt)
        ])
        
        # 构建中轴坐标系的正交基
        self._build_axis_basis()
    
    def _build_axis_basis(self):
        """构建中轴坐标系的正交基"""
        # Z' = 中轴方向
        self.z_prime = self.axis_dir
        
        # X' = 垂直于Z'且在XY平面内
        temp = np.array([0, 0, 1])
        self.x_prime = np.cross(temp, self.z_prime)
        self.x_prime /= np.linalg.norm(self.x_prime)
        
        # Y' = Z' × X'
        self.y_prime = np.cross(self.z_prime, self.x_prime)
    
    def to_axis_coords(self, P):
        """将标准坐标转换到中轴坐标系"""
        x, y, z = P
        x_p = np.dot(P, self.x_prime)
        y_p = np.dot(P, self.y_prime)
        z_p = np.dot(P, self.z_prime)
        
        r = np.sqrt(x_p**2 + y_p**2 + z_p**2)
        theta_p = np.arccos(z_p / r) if r > 0 else 0
        phi_p = np.arctan2(y_p, x_p)
        
        return r, theta_p, phi_p
    
    def yang_spiral(self, t):
        """阳螺旋参数方程"""
        r = 1 - t
        phi_p = 2 * np.pi * self.n_turns * t
        theta_p = np.pi/2 * (1 - self.k_pitch * t)
        return r, theta_p, phi_p
    
    def yin_spiral(self, t):
        """阴螺旋参数方程"""
        r = t
        phi_p = 2 * np.pi * self.n_turns * t + np.pi
        theta_p = np.pi/2 * (1 + self.k_pitch * t)
        return r, theta_p, phi_p
    
    def spiral_phase(self, P, tau):
        """计算螺旋相位"""
        r, theta_p, phi_p = self.to_axis_coords(P)
        
        y = np.sin(np.pi * tau)
        yi = np.cos(np.pi * tau)
        
        phase = y * np.cos(phi_p) + yi * np.sin(phi_p)
        phase *= (1 - abs(np.cos(theta_p)))  # 靠近中轴加权
        
        return phase
    
    def plot_spirals(self, n_points=100):
        """绘制双螺旋"""
        fig = plt.figure(figsize=(12, 10))
        ax = fig.add_subplot(111, projection='3d')
        
        # 阳螺旋
        t = np.linspace(0, 1, n_points)
        r_y, th_y, ph_y = zip(*[self.yang_spiral(ti) for ti in t])
        
        # 转换回直角坐标
        x_y = [ri * np.sin(thi) * np.cos(phi) for ri, thi, phi in zip(r_y, th_y, ph_y)]
        y_y = [ri * np.sin(thi) * np.sin(phi) for ri, thi, phi in zip(r_y, th_y, ph_y)]
        z_y = [ri * np.cos(thi) for ri, thi, phi in zip(r_y, th_y, ph_y)]
        
        ax.plot(x_y, y_y, z_y, 'r-', linewidth=2, label='阳螺旋')
        
        # 阴螺旋
        r_i, th_i, ph_i = zip(*[self.yin_spiral(ti) for ti in t])
        x_i = [ri * np.sin(thi) * np.cos(phi) for ri, thi, phi in zip(r_i, th_i, ph_i)]
        y_i = [ri * np.sin(thi) * np.sin(phi) for ri, thi, phi in zip(r_i, th_i, ph_i)]
        z_i = [ri * np.cos(thi) for ri, thi, phi in zip(r_i, th_i, ph_i)]
        
        ax.plot(x_i, y_i, z_i, 'b-', linewidth=2, label='阴螺旋')
        
        # 绘制球面
        u = np.linspace(0, 2*np.pi, 50)
        v = np.linspace(0, np.pi, 50)
        x_s = np.outer(np.cos(u), np.sin(v))
        y_s = np.outer(np.sin(u), np.sin(v))
        z_s = np.outer(np.ones(np.size(u)), np.cos(v))
        ax.plot_surface(x_s, y_s, z_s, alpha=0.1, color='gray')
        
        ax.set_xlabel('X')
        ax.set_ylabel('Y')
        ax.set_zlabel('Z')
        ax.legend()
        ax.set_title('后天八卦双螺旋球体')
        
        plt.show()

# 使用示例
if __name__ == "__main__":
    spiral = HoutianBaguaSpiralMath(axis_tilt=45.0)
    spiral.plot_spirals()
```
