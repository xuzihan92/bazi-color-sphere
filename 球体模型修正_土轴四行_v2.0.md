## 一、5/31 模型的问题

5/31 那晚我们把金（白）放在北极、水（黑/深蓝）放在南极。这是从「金=白」「水=黑」的文化标签硬套的，不是从日地关系推导的。

### 1.1 塌在哪里

- 金生水：从球顶沿经线下滑→纯白→浅蓝→深蓝→黑。**数学上能走，命理上牵强**——金（秋分·肃杀）怎么一步滑到水（冬至·极寒）？
- 火生土：红向球心走→降明度增暖度→沉淀为大地色。**但为什么火先到的是北极白极而不是球心的黄？**
- 水在球底（南极黑），金在球顶（北极白）——但火（夏至）和水（冬至）是黄道两端，应该是对跖点关系，不是极点关系。

### 1.2 漏了什么

把「黑」和「白」当成金和水的「本色」，而不是土的「极态」。黑白是**中轴线的两个端点**，不是黄道带上四行的属性。

## 二、梓涵的修正（2026-06-06 22:28）

> 原文摘录：
>
> "整颗球都是土。其他四行是为了标出土的东南西北。中轴线贯穿北极南极，这条轴就是土。黄道带绕在轴上，夏至在黄道面一端（火），冬至在另一端（水），春分秋分在黄道与赤道的交汇处（木和金）。"

### 2.1 新结构

```
        北极（白·阳之极）─ 也是土
              │
    夏至（火） ╱ ╲ 春分（木）
       ╱        ╲
      │   土(轴)  │  ← 整条中轴线都是土的本体
       ╲        ╱
    冬至（水） ╱ ╲ 秋分（金）
              │
        南极（黑·阴之极）─ 也是土
```

### 2.2 核心定义

| 概念 | 旧定义（5/31·错误） | 新定义（6/6·修正） |
|------|---------------------|---------------------|
| 土 | 球心一个灰点 | **整条中轴线**——从北极白贯穿球心到南极黑 |
| 四行 | 球面上五个色块 | 黄道带上的四个标记——夏至(火)、秋分(金)、冬至(水)、春分(木) |
| 北极 | 金（白） | **土**——阳之极，火生土的终点 |
| 南极 | 水（黑） | **土**——阴之极，水生土的终点 |
| 土的颜色 | 黄色（H=45°） | **明度轴本身**——从纯白到纯黑的整个灰度渐变 |
| 球体本质 | 五色拼成的球 | **整颗球就是土**，四行是黄道带在土球上的方位标记 |

## 三、生克在球体上的新走法

> 原文摘录（2026-06-06 22:53）：
>
> "火生土是光从夏至位置转向北极点的位置，而土生金是指由盛转衰，极点转向秋分。从火到金，刚好对应现实的夏至转向秋分的过程。"

### 3.1 生克路径

| 生克 | 旧走法（5/31） | 新走法（v2.0） | 物理对应 |
|------|----------------|----------------|------|
| 火生土 | 红→灰（向球心收束） | **火（夏至·黄道）→ 北极白（中轴上端）** 光走到极点，亮到极致转白 | 夏至→北极·极昼 |
| 土生金 | 灰→白（球心到球顶） | **北极白 → 秋分（金·黄道）** 由盛转衰，从轴线投射回黄道 | 北极→秋分 |
| 金生水 | 白→蓝→黑（球顶沿经线下滑） | **秋分（金）→ 冬至（水）** 黄道带上顺行，秋入冬 | 秋分→冬至 |
| 水生木 | 黑→蓝→绿（球底爬回赤道） | **冬至（水）→ 春分（木）** 黄道带上顺行，冬入春 | 冬至→春分 |
| 木生火 | 绿→红（赤道弧） | **春分（木）→ 夏至（火）** 黄道带上顺行，春入夏 | 春分→夏至 |
| 火克金 | — | **夏至（火）→ 秋分（金）** 北极端的克制关系 | 夏克秋 |
| 木克土 | — | **春分（木）→ 中轴土** 南极端的克制关系 | 春克中轴 |

### 3.2 南北极的不对称对称

> 原文摘录（2026-06-06 22:53）：
>
> "北极那端是夏至克秋分，中轴土生秋分。而南极是冬至生春分，春分克中轴。你说为啥看起来不对称，其实还是很对称的。只不过是阴阳上的对称，以及时间上的一致性。"

```
北极端（极昼·阳旺）:
    夏至(火) ──克──→ 秋分(金)
      │                  ↑
      生                 生
      ↓                  │
    中轴土 ──────────────┘
    （白·阳之极）

南极端（极夜·阴旺）:
    中轴土 ←──克── 春分(木)
      ↑                  ↑
      生                 生
      │                  │
    冬至(水) ──────────┘
    （黑·阴之极）
```

- 北极端：土在**生位**（生金），火在**克位**（克金）
- 南极端：土在**受克位**（被木克），水在**生位**（生木）
- 两端不是镜像——是**互补**。一端土主动，另一端土被动。
- 「极昼对极夜，生对克」——相位对称，不是几何对称。

## 四、四重自洽性验证

| 维度 | 如何自洽 |
|------|---------|
| **地日关系** | 黄赤交角 23.5°，黄道面与赤道面斜交。夏至/冬至在黄道两端，春分/秋分在黄赤交汇处——四行位置完全由地球轨道参数决定，不是任意贴标签 |
| **季节变化** | 火(夏至)→金(秋分)→水(冬至)→木(春分)→火——顺时针沿黄道带走一圈就是一年四季 |
| **五行生克** | 生克在球体上全通。火→土（收敛到极点）→金→水→木→土→火。没有断裂，没有需要「跳过」的 |
| **后天八卦** | 坎离对应子午（冬夏·水火），震兑对应卯酉（春秋·木金）。中轴线对应中央土 |
| **色彩对应** | 不是"火=红色"——是夏至的光谱色温天然偏暖红。冬至天然偏冷蓝。春分偏绿，秋分偏白。颜色从日地运动算出来，不是贴标签 |

## 五、对执中 v1.0 叠加规则的影响

### 5.1 致命的架构错误

执中的 `_wuxing_base_color()` 把土当成球面上第五个色相点（H=45° 暖黄），需要改为：

**土不是第五个算子——土是明度轴本身。**

### 5.2 需修改的核心项

| # | 问题 | v1.0 错误 | v2.0 正确做法 |
|----|------|-----------|---------------|
| 1 | 土的定义 | `_wuxing_base_color("土", i)` → H=45° | 土不存在于球面。土的"颜色"是**明度值 L**——0（南极黑）到 1（北极白） |
| 2 | 算子数量 | 五个算子（木火土金水） | **四个算子**（木火金水）。土不是算子——土是四行算子的**极值约束** |
| 3 | 火算子方向 | `"huo": 195.0`（青绿色，错的） | 火=夏至，在黄道面高点。HSL 中 0°（红）是合理的——夏至正午的天光色温 |
| 4 | 算子复合顺序 | 木→火→土→金→水（相生顺序串行） | 四行算子沿黄道面运行，**土算子只作用于极值**（当 S→0 或 L→极值时触发轴收敛） |
| 5 | 颜色空间 | HSL 柱坐标 | 仍然是 HSL，但 **H=黄道面方位角（春0°/夏90°/秋180°/冬270°），L=明度轴（土），S=距轴距离** |
| 6 | 五行扇区 | 五个扇区等分 | 黄道面四象限 + 中轴线（球心→两极） |

### 5.3 土算子的正确语义

```python
# 旧（错）：土是一个独立的颜色变换
土(color, i) → 降饱和、向灰偏

# 新（对）：土是"向中轴线收敛"的引力
土作用 = 任何颜色在 S→0 时汇聚到明度轴
       = 当四行算子的强度叠加到极端值时，自动向 L=0.5 收束
       = 土不是主动变换——是四行变换的边界条件
```

## 六、几何框架速查

```
球体坐标（黄道面 = 倾斜23.5°的环，中轴 = 垂直的明度轴）:

维度    │ 物理含义                │ 色彩对应
────────┼─────────────────────────┼────────────
H (色相) │ 黄道面方位角             │ 四行本色
         │ 0°=春分(木), 90°=夏至(火)│
         │ 180°=秋分(金), 270°=冬至(水)
────────┼─────────────────────────┼────────────
S (饱和度)│ 距中轴的距离              │ 偏离中轴的幅度
         │ S=1=黄道面纯色, S=0=轴上 │ → 土（灰）
────────┼─────────────────────────┼────────────
L (明度)  │ 沿中轴的高度              │ 土的本色
         │ L=1=北极白, L=0=南极黑   │ 阳→阴的全梯度
         │ L=0.5=赤道面 = 球心       │
```

## 七、元清的总结

> **这版修正把 5/31 那晚的「球体几何」从"艺术性强的隐喻模型"升级成了"在地球轨道力学上严格自洽的物理模型"。**
>
> 之前的漏洞（金生水为什么走那么远、火生土为什么先到白极、土为什么"被夹在中间"）全部消失——因为漏洞的根源就是把金水错放在了土的位置。
>
> 现在土回归到它本来的位置——那条贯穿南北的明度轴。四行回归到黄道带。生克是黄道带上的顺行和极点处的收敛/发散。颜色不是贴标签——是日地运动的自然结果。
>
> **下一步**：执中需按 v2.0 重写 `WuxingSphere` 底层——土不再是一个算子，土是明度轴和饱和度归零的引力。四行在黄道面上按春木-夏火-秋金-冬水（0°-90°-180°-270°）定位。算子强度需考虑八字权重在黄道面上的投影距离。
